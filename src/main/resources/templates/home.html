Como Funciona

Basta informar a URL do site que deseja clonar, escolher o modo de clonagem e a profundidade desejada. Nossa plataforma realiza a captura eficiente dos recursos, preservando links internos e configurando as permissões para links externos conforme sua necessidade.

Recursos Principais

  • Modos de clonagem: estático e headless
  • Configuração de profundidade para captura de links
  • Permissão para links externos para ampliar a captura
  • Download fácil do pacote clonado em ZIP
  • Interface moderna, responsiva e amigável
